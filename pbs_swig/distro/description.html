PBS programming with PYTHON

Introduction

This a pbs python module. So you can now use python instead of the PBS C API to make PBS programs to set/query queue, node and server attributes.

Installation

First you have to make the python module and test if it works. It depends on your python installation where you want to place the pbs modules. Some good places are:

/usr/lib/python1.5/site-packages
/usr/local/lib/python2.1/site-packages
